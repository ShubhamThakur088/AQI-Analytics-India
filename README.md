The AQI-Analytics-India project conducts an analytics of AQI across major Indian cities of 29 states. It helps get insights about the AQIs and their statuses, such as Good, Satisfactory, Moderate,
 Poor, Very Poor and Severe, from year 2022 to 2025. Along with statuses, it helps identify the prominent pollutant.

 AQI Ranges and their Status:
 0 - 50 -> Good
 51 - 100 -> Satisfactory
 101 - 200 -> Moderate
 201 - 300 -> Poor
 301 - 400 -> Very Poor
 401 - 500 -> Severe

 # Tools used
 - MS Excel
 - Tableau
 - Pandas and Numpy (Python)
